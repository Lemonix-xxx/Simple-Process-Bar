# Simple Process Bar

### 介绍
一个控制台自动生成进度条的工具

 *Version* : 0.1

### 文档

- :param char: 进度条填充字符

- :param bar_length: 进度条长度

- :param prompt: 提示符

- :param arrow: 箭头

- :param seconds: 进度变化的间隔

- :param side: 进度条两边的包裹字符

- :param change_length: 进度条是否变化长度

- :return: 无
```
                side[0]  side[1]
                   |        |
    Loading 37% >> [########]
    prompt      |    char
              arrow
```
